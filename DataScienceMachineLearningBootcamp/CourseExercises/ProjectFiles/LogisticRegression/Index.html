An Application

Overview

Information about the course and how this project got started.

In this project we will be working with a fake advertising data set, indicating whether or not a particular internet user clicked on an Advertisement on a company website. We will try to create a model that will predict whether or not they will click on an ad based off the features of that user. This data set contains 1,000 records, each containg the following fields:

  • 'Daily Time Spent on Site': consumer time on site in minutes
  • 'Age': cutomer age in years
  • 'Area Income': Avg. Income of geographical area of consumer
  • 'Daily Internet Usage': Avg. minutes a day consumer is on the internet
  • 'Ad Topic Line': Headline of the advertisement
  • 'City': City of consumer
  • 'Male': Whether or not consumer was male
  • 'Country': Country of consumer
  • 'Timestamp': Time at which consumer clicked on Ad or closed window
  • # * 'Clicked on Ad': 0 or 1 indicated clicking on Ad

Exploratory Data Analysis

The data, drawn from an e-commerce company, is part of the process to help determine the contributing factors leading to a website visitor clicking on the company's ad. In the sample data of 1,000, half clicked on the ad and half did not. Characteristics of the data set are summarized in the following table:
